# Hitachi Project

This repo contains code that was part of the Hitachi challenge during TAMU Datathon 2025. 

The challenge required us to leverage LLMs to automatically detect PII and other compliance issues in documents

## Run

Create a `credentials.py` file in the same directory as `main.py` that contains your Hugging Face API token

Then run

```python
python3 main.py
```
